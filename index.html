Keep up with the newest developments in the Solidity space.

The Solidity Summit is a free interactive forum with discussions and talks on Solidity, Yul, language design, security and tooling.

During the event you can expect to:

  • get up to speed with latest language proposals and new features.
  • hear updates from Solidity tooling and security experts.
  • learn from Solidity power users.

The event wants to promote useful discussions that result in language improvement proposals, leading to actual implementations. Furthermore, we aim to foster communication between teams working on similar topics and identify needs for the smart contract ecosystem for Ethereum.

Solidity Summit 2022 is part of Devconnect Amsterdam and will be a one-day forum for the Solidity ecosystem. It's for all people involved and interested in the Solidity language and the tooling around it.

Who can join?

The Solidity Summit focuses on advanced Solidity users and other stakeholders of the Solidity ecosystem, such as tool builders, auditors and security researchers.

Solidity Language Shaper

Do you want to elaborate on a change proposal for the Solidity language or are you using a language feature in a novel way that might benefit others? Then consider applying for a talk.

Tooling Builders & Auditors

Did you build a useful tool you want to share with the audience? Are you missing specific compiler output? Are you a Solidity security expert? Consider applying for a talk to share your insights.

Solidity Enthusiasts

We welcome all Solidity power users and code aficionados to join, learn, network and share their thoughts and feedback.

2022 Topics

This year, presentations and discussions will evolve around the following topics:

  • Solidity and Yul
  • Language changes or extension proposals
  • Upcoming new features
  • Feedback on recently introduced features
  • Unification / extension of the ABI
  • Tooling, debuggers and debuggin data output
  • Testing frameworks
  • Integration into existing "larger ecosystem tools" (e.g. vscode via Language Server, LLVM, DWARF)
  • Source verification
  • Layer 2 support, cross-chain support
  • Mechanisms / programming patterns (on the EVM) like mutexes, state machines, upgradeable contracts, libraries, create2, etc.

Looking forward to seeing you there!

To participate in the Solidity Summit, apply for a speaking slot or fill in the form for attendees. Since we can only give out a limited amount of tickets, please use the "participate" form to describe why this event is relevant for you. Participation is free.
